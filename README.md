# Student Dashboard

This repository contains a Streamlit dashboard for visualising student information exported from a Django-based student management system.

## Files used

The dashboard reads the following Excel files from the `data/` folder:

- `preprocess_students.xlsx`
- `registered_students.xlsx`
- `graduated_students.xlsx`
- `supervisor_student_report.xlsx`
- `external_examiner_report.xlsx`

## Run locally

Create a virtual environment, install the requirements, and run:

pip install -r requirements.txt
python runfile.py
